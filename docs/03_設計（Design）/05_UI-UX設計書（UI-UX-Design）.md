## 1. デザイン方針

### 1.1 基本方針

| 方針 | 説明 |
|------|------|
| Material Design 3準拠 | Vuetify 3によるMaterial Designコンポーネントの統一使用 |
| データ重視 | GRCデータの視覚的な表現を重視（チャート、ゲージ、ヒートマップ） |
| 効率性 | 3クリック以内で主要機能にアクセス可能 |
| 一貫性 | 全画面でUIパターン・用語・操作性を統一 |
| フィードバック | 操作結果をリアルタイムにフィードバック（トースト通知） |
| エラー防止 | 確認ダイアログ、バリデーション、ヘルプの提供 |

### 1.2 技術スタック

| 要素 | 技術 |
|------|------|
| UIフレームワーク | Vue.js 3（Composition API） |
| UIライブラリ | Vuetify 3（Material Design） |
| 状態管理 | Pinia |
| ルーティング | Vue Router 4 |
| チャート | Chart.js / vue-chartjs |
| アイコン | Material Design Icons (mdi) |
| 型言語 | TypeScript |
