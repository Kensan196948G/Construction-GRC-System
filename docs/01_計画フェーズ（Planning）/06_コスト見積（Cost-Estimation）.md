## 7. コスト管理方針

### 7.1 予算管理ルール

| ルール | 内容 |
|--------|------|
| 予算消化率の監視 | 月次で計画値vs実績値を比較（EVM手法） |
| 予算超過アラート | 計画値の105%到達時点でPMに警告 |
| 予算超過承認 | 10%以内はPM承認、10%超はSC承認 |
| 予備費利用 | コンティンジェンシーはPM承認、マネジメントリザーブはSC承認 |
| コスト報告 | 月次SC報告に予算消化状況を含める |

### 7.2 EVM（アーンドバリューマネジメント）指標

| 指標 | 定義 | 管理基準 |
|------|------|----------|
| CPI（コスト効率指数） | EV / AC | 0.9以上を維持 |
| SPI（スケジュール効率指数） | EV / PV | 0.9以上を維持 |
| EAC（完了時総コスト見積） | BAC / CPI | 予算の110%以内 |
| TCPI（残作業効率指数） | (BAC-EV) / (BAC-AC) | 1.1以下を維持 |
