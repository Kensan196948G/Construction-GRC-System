## 10. クリティカルパス（プロジェクト全体）

プロジェクト全体のクリティカルパスは以下の通りである。

```
M1 → Phase1(1.1→1.2→1.3→1.8→1.9) → Phase2(2.1→2.3→2.4→2.6)
    → Phase3(3.1→3.5→3.6) → Phase4(4.1→4.3→4.4→4.6)
    → Phase5(5.1→5.2→5.5→5.6) → 運用移行(6.1→6.2→6.3) → M9
```

**クリティカルパス上の総工数：約1,100人日**
**クリティカルパス上の所要期間：約16ヶ月**
