## 1. 概要

本文書は、建設業統合GRCシステム構築プロジェクトに関わる全てのステークホルダーを識別し、各ステークホルダーの影響度・関心度を分析した上で、適切なコミュニケーション方針を定義する。

## 3. 影響度・関心度マトリクス

### 3.1 評価基準

**影響度（Power）：** プロジェクトに対する意思決定権限や影響力

| レベル | スコア | 定義 |
|--------|--------|------|
| 最高 | 5 | プロジェクトの継続/中止を判断できる |
| 高 | 4 | 主要な意思決定に関与する |
| 中 | 3 | 特定領域の意思決定に関与する |
| 低 | 2 | 限定的な影響力を持つ |
| 最低 | 1 | 直接的な影響力はない |

**関心度（Interest）：** プロジェクトに対する関心・関与の度合い

| レベル | スコア | 定義 |
|--------|--------|------|
| 最高 | 5 | 日常的にプロジェクトに関与している |
| 高 | 4 | 定期的にプロジェクト状況を確認している |
| 中 | 3 | 重要な局面でプロジェクトに関与する |
| 低 | 2 | プロジェクトへの関心は限定的 |
| 最低 | 1 | プロジェクトへの関心がほとんどない |

## 4. ステークホルダー別対応方針

### 4.1 緊密に管理（Manage Closely）

対象：S02（CIO）、S03（管理本部長）、S06（情報システム部長）、S07（PM）、S21（SIベンダーPM）

| 対応策 | 内容 |
|--------|------|
| コミュニケーション頻度 | 週次以上（S02,S03は月次SC＋隔週個別報告） |
| 情報提供レベル | 詳細な進捗・課題・リスク・予算情報 |
| 意思決定への関与 | 全ての重要意思決定に参画 |
| 関係構築 | 定期的な1on1ミーティング、非公式な情報共有 |
| 期待値管理 | プロジェクト計画時に期待値を合意、変更時は事前に説明 |

### 4.2 満足を維持（Keep Satisfied）

対象：S01（社長）、S04（土木本部長）、S05（建築本部長）、S25（外部監査法人）、S26（ISO認証機関）

| 対応策 | 内容 |
|--------|------|
| コミュニケーション頻度 | 月次〜四半期 |
| 情報提供レベル | サマリーレベル（経営ダッシュボード、1枚報告） |
| 意思決定への関与 | 重大な意思決定のみ（予算超過、スケジュール大幅変更等） |
| 関係構築 | SC報告を通じた定期的な接点確保 |
| 期待値管理 | 成果物のデモによる可視化、ROI進捗の報告 |

### 4.3 情報提供（Keep Informed）

対象：S08-S12（部門長）、S15（支店長）、S17-S20（担当者・運用チーム）

| 対応策 | 内容 |
|--------|------|
| コミュニケーション頻度 | 月次（部門長）、スプリント毎（担当者） |
| 情報提供レベル | 関連機能の進捗、導入スケジュール、教育計画 |
| 意思決定への関与 | 該当領域の業務要件確認、UAT参加 |
| 関係構築 | デモ会への招待、フィードバック収集 |
| 期待値管理 | 早期からの参画による当事者意識の醸成 |

### 4.4 監視（Monitor）

対象：S13（人事部長）、S14（総務部長）、S16（現場所長）

| 対応策 | 内容 |
|--------|------|
| コミュニケーション頻度 | 四半期（ニュースレター等） |
| 情報提供レベル | プロジェクトの概況のみ |
| 意思決定への関与 | 必要に応じて意見照会 |
| 関係構築 | プロジェクトニュースレターの配信 |
| 期待値管理 | 全社周知を通じた基本的な認知確保 |

## 6. ステークホルダーエンゲージメント計画

### 6.1 エンゲージメントレベル定義

| レベル | 定義 |
|--------|------|
| 抵抗（Resistant） | プロジェクトに反対している |
| 無関心（Unaware） | プロジェクトを認知していない |
| 中立（Neutral） | プロジェクトを認知しているが支持も反対もしていない |
| 支持（Supportive） | プロジェクトを支持し、協力的である |
| 推進（Leading） | プロジェクトを積極的に推進している |

### 6.2 エンゲージメント目標

| ID | ステークホルダー | 現状レベル | 目標レベル | ギャップ | 施策 |
|----|----------------|-----------|-----------|---------|------|
| S01 | 社長 | 支持 | 支持 | なし | 定期報告の継続 |
| S02 | CIO | 推進 | 推進 | なし | 密なコミュニケーション継続 |
| S03 | 管理本部長 | 支持 | 推進 | あり | コンプライアンス改善効果の可視化 |
| S04 | 土木本部長 | 中立 | 支持 | あり | 土木事業部向けデモ、メリット説明 |
| S05 | 建築本部長 | 中立 | 支持 | あり | 建築事業部向けデモ、メリット説明 |
| S08 | 安全環境部長 | 支持 | 推進 | あり | 安全管理効率化の具体的効果提示 |
| S10 | 内部監査室長 | 支持 | 推進 | あり | 監査工数削減の数値的効果提示 |
| S15 | 各支店長 | 無関心 | 支持 | あり | 支店向け説明会の実施 |
| S16 | 現場所長 | 無関心 | 中立 | あり | 現場負担軽減のメリット説明 |
| S17 | リスク管理担当者 | 中立 | 推進 | あり | 早期参画、キーユーザー任命 |

### 6.3 チェンジマネジメント施策

| # | 施策 | 対象 | 時期 | 内容 |
|---|------|------|------|------|

## 7. モニタリングと見直し

### 7.1 モニタリング方法

| 活動 | 頻度 | 担当 | 内容 |
|------|------|------|------|
| ステークホルダー満足度調査 | 四半期 | PMO | アンケートによる満足度・要望の収集 |
| エンゲージメントレベル評価 | 月次 | PM | 各ステークホルダーのエンゲージメントレベル再評価 |
| コミュニケーション効果測定 | 月次 | PMO | 会議出席率、フィードバック件数の分析 |
| 新規ステークホルダーの識別 | 随時 | PM | 組織変更、スコープ変更に伴う新規ステークホルダーの追加 |

### 7.2 見直しトリガー

| トリガー | 対応 |
|---------|------|
| 組織変更・人事異動 | ステークホルダー一覧の更新、新任者への説明 |
| スコープ変更 | 影響を受けるステークホルダーの再評価 |
| エンゲージメントレベルの低下 | 個別対応策の策定・実施 |
| 新たな懸念事項の発生 | 対応方針の追加・更新 |
