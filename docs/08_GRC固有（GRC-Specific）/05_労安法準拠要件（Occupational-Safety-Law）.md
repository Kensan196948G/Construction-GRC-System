## 1. 目的

本文書は、労働安全衛生法（労安法）の建設現場における安全管理要件に基づくシステム対応要件を定義する。建設現場の安全衛生管理、安全衛生計画の策定・実行・記録をConstruction-GRC-Systemで管理し、労働者の安全と健康を確保することを目的とする。

---

## 2. 労働安全衛生法の概要

### 2.1 法律の目的（第1条）

労働災害の防止のための危害防止基準の確立、責任体制の明確化及び自主的活動の促進の措置を講ずる等その防止に関する総合的計画的な対策を推進することにより、職場における労働者の安全と健康を確保するとともに、快適な職場環境の形成を促進する。

### 2.2 建設業における適用の特徴

| 特徴 | 説明 | システム対応 |
|------|------|------------|
| 元方事業者の義務 | 特定元方事業者（元請業者）に対する統括安全衛生管理義務 | 統括安全衛生管理機能 |
| 重層下請構造 | 複数の下請業者が同一現場で作業 | 下請業者安全管理連携 |
| 移動する職場 | 建設現場は工事の進捗に応じて変化 | 現場別安全管理対応 |
| 危険作業の多さ | 高所作業、重機使用、掘削作業等 | 危険作業管理、特別教育管理 |
| 有期的な事業 | 工事完了と共に事業が終了 | 工事単位の安全管理記録 |

## 3. 安全衛生管理体制（第10条〜第19条の3）

### 3.1 管理者の選任義務

| 管理者 | 選任要件 | 職務 | システム管理要件 |
|--------|---------|------|---------------|
| 総括安全衛生管理者 | 常時100人以上の労働者を使用する事業場 | 安全衛生業務の統括管理 | 選任記録、届出管理 |
| 安全管理者 | 常時50人以上の労働者を使用する事業場 | 安全に関する技術的事項の管理 | 資格確認、選任記録 |
| 衛生管理者 | 常時50人以上の労働者を使用する事業場 | 衛生に関する技術的事項の管理 | 資格確認、選任記録 |
| 安全衛生推進者 | 常時10人以上50人未満の労働者を使用する事業場 | 安全衛生に関する業務の推進 | 選任記録 |
| 産業医 | 常時50人以上の労働者を使用する事業場 | 労働者の健康管理 | 契約管理、面接指導記録 |
| 作業主任者 | 政令で定める危険作業を行う場合 | 当該作業の安全管理 | 資格管理、配置管理 |

### 3.2 建設現場特有の管理体制

| 管理者 | 選任要件（建設業） | システム管理要件 |
|--------|-----------------|---------------|
| 統括安全衛生責任者 | 特定元方事業者で常時50人以上（ずい道等は30人以上） | 選任・届出管理、権限範囲管理 |
| 元方安全衛生管理者 | 統括安全衛生責任者を選任した事業場 | 資格確認、選任記録 |
| 安全衛生責任者 | 統括安全衛生責任者を選任すべき事業場の関係請負人 | 下請業者の安全衛生責任者管理 |
| 店社安全衛生管理者 | 一定規模以上の建設工事（20億円以上等） | 巡回記録管理、指導記録 |

## 5. 危険防止措置（第20条〜第36条）

### 5.1 事業者の講ずべき措置

| 措置区分 | 条文 | 対象リスク | システム管理要件 |
|---------|------|----------|---------------|
| 危険防止措置 | 第20条 | 機械、爆発物、電気等による危険 | リスクアセスメント記録、対策管理 |
| 健康障害防止措置 | 第22条 | 有害物質、放射線、高温等 | 有害業務管理、暴露記録 |
| 作業環境管理 | 第22条 | 作業場の環境条件 | 環境測定記録管理 |
| 作業行動管理 | 第24条 | 労働者の作業行動に起因する災害 | 不安全行動管理 |

### 5.2 建設業における主要な危険作業管理

| 危険作業 | 根拠条文 | 必要な資格・教育 | システム管理要件 |
|---------|---------|---------------|---------------|
| 高所作業（2m以上） | 安衛則第518条〜 | 足場の組立て等作業主任者 | 足場点検記録、作業主任者配置管理 |
| クレーン作業 | クレーン等安全規則 | クレーン運転士免許、玉掛け技能講習 | 資格確認、作業計画管理 |
| 掘削作業（2m以上） | 安衛則第355条〜 | 地山の掘削作業主任者 | 地山調査記録、作業主任者配置 |
| 型枠支保工 | 安衛則第239条〜 | 型枠支保工の組立て等作業主任者 | 組立図管理、点検記録 |
| 鉄骨建方作業 | 安衛則第517条の2〜 | 建築物等の鉄骨の組立て等作業主任者 | 作業計画管理、点検記録 |
| 解体工事 | 安衛則第517条の14〜 | コンクリート造の工作物の解体等作業主任者 | 解体計画管理、アスベスト調査記録 |
| 酸素欠乏危険作業 | 酸欠則 | 酸素欠乏危険作業主任者 | 酸素濃度測定記録、換気管理 |
| 有機溶剤作業 | 有機則 | 有機溶剤作業主任者 | 作業環境測定記録、健康診断管理 |
| 石綿（アスベスト）作業 | 石綿則 | 石綿作業主任者 | 事前調査記録、暴露防止措置管理 |

### 5.3 リスクアセスメント管理機能

| 機能 | 詳細 |
|------|------|
| 危険性・有害性の特定 | 作業別の危険要因データベース、チェックリスト |
| リスクの見積り | 頻度×重篤度のマトリクス評価、リスクレベル自動判定 |
| リスク低減措置 | 本質的安全対策→工学的対策→管理的対策→個人用保護具の優先順位管理 |
| KY（危険予知）活動記録 | 日次KYシート管理、危険ポイントの記録 |
| ヒヤリハット管理 | ヒヤリハット報告、統計分析、傾向分析 |
| 作業手順書管理 | 危険作業の手順書テンプレート、版管理 |

## 7. 健康管理（第66条〜第71条）

### 7.1 健康診断の種類と管理

| 健康診断種類 | 対象者 | 実施頻度 | システム管理 |
|------------|--------|---------|------------|
| 一般健康診断（雇入時） | 常時使用する労働者 | 雇入れ時 | 受診記録、結果管理 |
| 一般健康診断（定期） | 常時使用する労働者 | 1年以内ごと | 受診スケジュール、結果管理 |
| 特殊健康診断 | 有害業務従事者 | 6ヶ月以内ごと | 対象者抽出、受診管理 |
| じん肺健康診断 | 粉じん作業従事者 | 管理区分に応じて | 対象者管理、経過管理 |
| 石綿健康診断 | 石綿取扱い業務従事者 | 6ヶ月以内ごと | 対象者管理、暴露記録連携 |
| 長時間労働者面接指導 | 月80時間超の時間外労働者 | 該当時 | 労働時間管理との連携 |
| ストレスチェック | 常時50人以上の事業場の労働者 | 1年以内ごと | 実施記録、高ストレス者対応 |

### 7.2 健康管理機能

| 機能 | 詳細 |
|------|------|
| 健診スケジュール管理 | 対象者別の健診スケジュール、受診率管理 |
| 結果管理 | 健診結果の記録（法定保存期間：5年/30年/40年） |
| 有所見者管理 | 有所見者のフォローアップ、就業制限管理 |
| 面接指導管理 | 長時間労働者・高ストレス者の面接指導記録 |
| 化学物質暴露管理 | 有害物質暴露量の記録、濃度測定記録 |

## 8. 建設現場特有の安全管理

### 8.1 安全パトロール管理

| パトロール種別 | 頻度 | 実施者 | システム管理 |
|-------------|------|--------|------------|
| 日常パトロール | 毎日 | 安全衛生責任者 | チェックリスト、是正指示管理 |
| 週間パトロール | 週1回 | 統括安全衛生責任者 | パトロール記録、写真管理 |
| 本社パトロール | 月1〜2回 | 店社安全衛生管理者 | 指摘事項管理、改善追跡 |
| 発注者パトロール | 適宜 | 発注者監督員 | 指摘事項受理、対応記録 |

### 8.2 安全衛生協議会管理

| 管理項目 | 詳細 |
|---------|------|
| 開催管理 | 月1回以上の定期開催、議事録管理 |
| 出席管理 | 元請・全下請の安全衛生責任者の出席管理 |
| 議題管理 | 安全衛生活動報告、災害事例共有、翌月計画 |
| 決定事項追跡 | 協議会で決定した事項の実施状況追跡 |

### 8.3 労働災害報告管理

| 報告種別 | 報告義務 | 期限 | システム管理 |
|---------|---------|------|------------|
| 死亡・重傷（休業4日以上） | 労働者死傷病報告（様式第23号） | 遅滞なく | 報告書自動生成、提出管理 |
| 軽傷（休業4日未満） | 労働者死傷病報告（様式第24号） | 四半期ごと | 集計レポート、提出管理 |
| 重大災害 | 事故報告 | 直ちに | 緊急報告フロー、当局通知 |

### 8.4 災害統計管理

| 指標 | 計算式 | システム管理 |
|------|--------|------------|
| 度数率 | (死傷者数 / 延労働時間数) × 1,000,000 | 自動計算、推移グラフ |
| 強度率 | (労働損失日数 / 延労働時間数) × 1,000 | 自動計算、推移グラフ |
| 年千人率 | (死傷者数 / 平均労働者数) × 1,000 | 自動計算、業界比較 |
| 無災害記録 | 無災害継続日数 | 自動カウント、表示 |
