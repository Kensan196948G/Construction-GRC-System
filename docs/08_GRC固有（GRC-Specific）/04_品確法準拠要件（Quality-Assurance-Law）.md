## 1. 目的

本文書は、「公共工事の品質確保の促進に関する法律」（品確法）の要件に基づくシステム対応要件を定義する。技術提案・評価制度、総合評価落札方式等への対応をConstruction-GRC-Systemで管理し、公共工事の品質確保に貢献することを目的とする。

---

## 2. 品確法の概要

### 2.1 法律の目的（第1条）

公共工事の品質は、現在及び将来の国民の安全と生活の質の向上に寄与するものであることに鑑み、公共工事の品質確保に関する施策を総合的に推進し、もって国民の福祉の向上及び国民経済の健全な発展に寄与する。

### 2.2 基本理念（第3条）

| 基本理念 | システムでの対応 |
|---------|---------------|
| 価格と品質が総合的に優れた調達 | 総合評価落札方式対応、技術提案管理 |
| 公共工事の担い手の中長期的な育成確保 | 技術者育成管理、キャリアパス管理 |
| 災害時の迅速な対応 | 災害対応体制管理、防災協定管理 |
| 多様な入札契約方式の活用 | 入札方式別対応機能 |
| 適正な利潤の確保 | 適正価格管理、見積チェック機能 |
| 働き方改革への対応 | 労働時間管理、週休二日制対応 |

### 2.3 品確法の改正経緯

| 改正年 | 主な改正内容 | システム対応 |
|--------|------------|------------|
| 2005年 | 品確法制定（総合評価方式の導入推進） | 総合評価管理の基本機能 |
| 2014年改正 | 担い手の中長期的な育成確保、ダンピング対策、多様な入札契約方式 | 担い手確保管理機能、ダンピング検知 |
| 2019年改正 | 働き方改革、災害対応、i-Construction推進 | 労働環境管理、災害対応管理、ICT活用管理 |

## 3. 発注者の責務と受注者の対応

### 3.1 発注者の責務（第7条〜第14条）

| 責務 | 条文 | システム対応要件 |
|------|------|---------------|
| 仕様書・設計書の適切な作成 | 第7条の2 | 発注情報管理、設計図書チェック |
| 適切な予定価格の設定 | 第7条の3 | 予定価格分析、適正価格チェック |
| 適切な工期の設定 | 第7条の4 | 工期算定支援、適正工期チェック |
| 適切な設計変更 | 第7条の5 | 設計変更管理、変更理由記録 |
| 適切な情報の公表 | 第14条 | 公表情報管理 |

### 3.2 受注者の責務とシステム対応

| 責務 | 条文 | システム対応要件 |
|------|------|---------------|
| 適正な施工 | 第8条 | 品質管理記録、施工記録管理 |
| 技術提案の適切な実施 | 第8条の2 | 技術提案管理、実施状況追跡 |
| 技術者の育成確保 | 第8条の3 | 技術者育成計画管理、CPD記録 |
| 下請契約の適正化 | 第8条の4 | 下請管理、契約条件チェック |

## 4. 総合評価落札方式への対応

### 4.1 総合評価落札方式の類型

| 類型 | 適用工事 | 評価項目 | システム対応 |
|------|---------|---------|------------|
| 簡易型 | 一般的な工事 | 企業の施工実績、配置技術者の実績 | 実績データの管理・検索・出力 |
| 標準型 | 技術的な工夫が必要な工事 | 技術提案（施工計画、技術的所見） | 技術提案書作成支援、評価項目管理 |
| 高度技術提案型 | 高度な技術力を要する工事 | 詳細な技術提案、VE提案 | 高度技術提案管理、VE分析支援 |

## 5. 工事成績評定への対応

### 5.1 工事成績評定の構成

| 評定区分 | 配点 | システム管理要件 |
|---------|------|---------------|
| 施工体制（施工体制一般・配置技術者） | 各項目1〜5点 | 施工体制台帳管理、技術者配置管理 |
| 施工状況（施工管理・工程管理・安全対策・対外関係） | 各項目1〜5点 | 施工記録管理、日報管理、安全管理記録 |
| 出来形及び出来ばえ（品質・出来ばえ） | 各項目1〜5点 | 品質管理記録、出来形管理データ |
| 創意工夫・社会性等 | 加点項目 | 技術提案実施記録、社会貢献記録 |
| 法令遵守等 | 減点項目 | 法令違反記録、指導事項管理 |

### 5.2 工事成績管理機能

| 機能 | 詳細 |
|------|------|
| 評定点記録 | 工事別の評定点記録、内訳管理 |
| 統計分析 | 平均点推移、業種別分析、発注者別分析 |
| 改善管理 | 低評価項目の分析、改善措置計画 |
| ベンチマーク | 業界平均・目標値との比較 |
| アラート | 目標値を下回る場合の警告 |

## 6. 多様な入札契約方式

### 6.1 入札契約方式の分類

| 方式 | 説明 | システム対応 |
|------|------|------------|
| 価格競争方式 | 価格のみで落札者を決定 | 入札情報管理、見積管理 |
| 総合評価落札方式 | 価格と品質を総合的に評価 | 技術提案管理、評価点管理 |
| 技術提案・交渉方式 | 最も優れた技術提案を行った者と交渉 | 技術提案管理、交渉記録管理 |
| ECI方式 | 設計段階から施工者が参画 | 設計協力記録管理 |
| CM方式 | コンストラクションマネージャーが関与 | CM関連記録管理 |
| 設計施工一括発注 | 設計と施工を一括して発注 | 統合プロジェクト管理 |

## 7. 担い手確保・育成への対応

### 7.1 担い手確保の取り組み管理

| 取り組み | 品確法条文 | システム管理要件 |
|---------|----------|---------------|
| 適正な利潤の確保 | 第3条第8項 | 工事別収支管理、利益率分析 |
| 適正な工期の確保 | 第7条の4 | 工期管理、週休二日実施率管理 |
| 技術者の育成 | 第8条の3 | 教育訓練計画管理、CPD管理、資格取得支援管理 |
| 若年技術者の確保 | 基本方針 | 若年者採用・定着率管理、キャリアパス管理 |
| 働き方改革 | 2019年改正 | 労働時間管理、有給取得率管理、週休二日実施管理 |

### 7.2 技術者育成管理機能

| 機能 | 詳細 |
|------|------|
| キャリアパス管理 | 技術者の成長ロードマップ管理 |
| 資格取得支援 | 取得すべき資格の推奨、受験スケジュール管理 |
| OJT管理 | 現場配置によるOJT計画・実績管理 |
| CPD管理 | 継続教育の受講記録、単位集計 |
| CCUS連携 | 建設キャリアアップシステムとの就業履歴連携 |
| 表彰管理 | 優良技術者・優良現場代理人の表彰記録 |

## 8. i-Construction対応

### 8.1 ICT活用実績管理

| ICT技術 | 管理項目 | システム対応 |
|---------|---------|------------|
| ICT土工 | 3次元測量・設計・施工・検査 | ICT活用実績記録、CIM/BIMデータ管理 |
| UAV（ドローン） | 測量・写真撮影・点検 | UAV活用記録、飛行計画管理 |
| 3次元計測 | レーザースキャナー、写真測量 | 計測データ管理 |
| BIM/CIM | 3次元モデルの活用 | BIM/CIMモデル管理、活用実績記録 |
| AI・IoT | 安全管理、品質管理 | AI/IoT活用実績記録 |

### 8.2 ICT活用評価管理

| 評価項目 | 管理内容 |
|---------|---------|
| ICT活用工事の実績数 | 工事種別別のICT活用実績カウント |
| ICT技術の習熟度 | 技術者ごとのICTスキルレベル管理 |
| 生産性向上効果 | ICT活用による省力化・効率化の定量データ |
| 加点実績 | 総合評価における加点実績の記録 |
