## 1. 目的

本文書は、ISO/IEC 27001:2022 附属書Aに定義された全93管理策と、Construction-GRC-Systemのシステム機能とのマッピングを定義する。各管理策のシステム実装状況、対応機能、実装優先度を明確化し、情報セキュリティマネジメントシステム（ISMS）の効果的な運用を支援することを目的とする。

---

## 2. 管理策の概要

### 2.1 ISO/IEC 27001:2022 管理策構成

| ドメイン | カテゴリ | 管理策数 |
|---------|---------|---------|
| A.5 | 組織的管理策（Organizational Controls） | 37 |

| A.8 | 技術的管理策（Technological Controls） | 34 |

### 2.2 実装優先度の定義

| 優先度 | 説明 | 実装時期 |
|--------|------|---------|
| P1（必須） | システム運用開始に必須の管理策 | Phase 1（v1.0.0） |
| P2（重要） | 初期運用後に必要な管理策 | Phase 1.x（v1.1.0〜v1.3.0） |
| P3（推奨） | 成熟度向上のために推奨される管理策 | Phase 2（v2.0.0） |
| P4（将来） | 将来的に対応を検討する管理策 | Phase 3以降 |

### 2.3 実装ステータスの定義

| ステータス | 説明 |
|-----------|------|
| 実装済み | システム機能として実装完了 |
| 部分実装 | 一部機能が実装済み、完全対応は未了 |
| 計画中 | 実装計画があり開発予定 |
| 運用対応 | システム機能ではなく運用ルールで対応 |
| 対象外 | 建設業GRCシステムの適用範囲外 |
