## 1. 目的

本文書は、建設業法の主要条項に基づくシステム対応要件を定義する。建設業許可要件、経営事項審査、下請契約規制等の法定要件をConstruction-GRC-Systemで管理し、建設業者としてのコンプライアンスを確保することを目的とする。

---

## 2. 建設業法の概要

### 2.1 法律の目的（第1条）

建設業を営む者の資質の向上、建設工事の請負契約の適正化等を図ることにより、建設工事の適正な施工を確保し、発注者を保護するとともに、建設業の健全な発達を促進し、もって公共の福祉の増進に寄与する。

### 2.2 適用範囲

| 対象 | 説明 |
|------|------|
| 建設業者 | 建設業の許可を受けて建設業を営む者 |
| 建設工事 | 土木建築に関する工事で建設業法別表に掲げるもの（29業種） |
| 建設業の許可 | 国土交通大臣許可 / 都道府県知事許可 |

### 2.3 建設業29業種

| 区分 | 業種名 |
|------|--------|
| 総合建設業（2業種） | 土木工事業、建築工事業 |
| 専門建設業（27業種） | 大工工事業、左官工事業、とび・土工工事業、石工事業、屋根工事業、電気工事業、管工事業、タイル・れんが・ブロック工事業、鋼構造物工事業、鉄筋工事業、舗装工事業、しゅんせつ工事業、板金工事業、ガラス工事業、塗装工事業、防水工事業、内装仕上工事業、機械器具設置工事業、熱絶縁工事業、電気通信工事業、造園工事業、さく井工事業、建具工事業、水道施設工事業、消防施設工事業、清掃施設工事業、解体工事業 |

## 3. 建設業許可要件（第3条〜第17条）

### 3.1 許可区分

| 許可区分 | 条件 | システム管理要件 |
|---------|------|---------------|
| 大臣許可 | 2以上の都道府県に営業所を設置 | 許可区分の管理、営業所一覧管理 |
| 知事許可 | 1つの都道府県内のみに営業所を設置 | 許可区分の管理、営業所管理 |
| 特定建設業 | 発注者から直接請け負った工事で下請代金総額4,500万円以上（建築は7,000万円以上） | 許可種別の管理、下請金額の自動チェック |
| 一般建設業 | 上記以外 | 許可種別の管理 |

### 3.2 許可要件（第7条・第15条）

| 要件 | 条文 | システム対応要件 |
|------|------|---------------|
| 経営業務の管理責任者 | 第7条第1号 | 経営業務管理責任者の登録・資格確認・在籍管理 |
| 専任技術者 | 第7条第2号、第15条第2号 | 技術者の資格情報管理、配置状況管理、専任性チェック |
| 財産的基礎・金銭的信用 | 第7条第4号、第15条第3号 | 財務情報管理、自己資本額・流動比率チェック |
| 欠格要件に該当しないこと | 第8条 | 欠格要件チェックリスト管理 |
| 誠実性 | 第7条第3号 | 不正行為履歴管理 |

## 4. 経営事項審査（第27条の23〜第27条の29）

### 4.1 経営事項審査の概要

公共工事を直接請け負おうとする建設業者は、経営事項審査（経審）を受ける必要がある。

### 4.2 審査項目とシステム管理

| 審査項目 | 評価基準 | システム管理要件 |
|---------|---------|---------------|
| 経営規模（X1） | 完成工事高 | 工種別完成工事高の記録・集計・推移管理 |
| 経営規模（X2） | 自己資本額、利払前税引前償却前利益 | 財務データの管理・分析 |
| 技術力（Z） | 技術職員数、元請完成工事高 | 技術者の資格・人数管理、元請工事実績管理 |
| その他の審査項目（社会性等）（W） | 労働福祉、建設業の営業年数、防災協定、法令遵守、建設機械保有状況、ISO認証等 | 各種実績データの統合管理 |
| 経営状況（Y） | 純支払利息比率、負債回転期間、売上高経常利益率等 | 経営状況分析データの管理 |

### 4.4 総合評定値（P点）の計算式

```
P = 0.25×X1 + 0.15×X2 + 0.20×Y + 0.25×Z + 0.15×W

X1: 経営規模（完成工事高）           0〜2,309
X2: 経営規模（自己資本額、利益額）    0〜2,280
Y:  経営状況                         0〜1,595
Z:  技術力                           0〜2,441
W:  その他の審査項目（社会性等）      0〜1,966
```

## 5. 請負契約の規制（第18条〜第24条の8）

### 5.1 請負契約の原則

| 条文 | 要件 | システム対応要件 |
|------|------|---------------|
| 第18条 | 建設工事の請負契約の原則（対等な立場、合意に基づく公正な契約） | 契約管理モジュール、契約条件チェック |
| 第19条 | 建設工事の請負契約の内容（書面主義、14項目の記載義務） | 契約書テンプレート、必須項目チェック機能 |
| 第19条の3 | 不当に低い請負代金の禁止 | 適正価格チェック、警告機能 |
| 第19条の4 | 不当な使用資材等の購入強制の禁止 | 資材購入条件管理 |
| 第19条の5 | 著しく短い工期の禁止 | 工期適正性チェック機能 |

### 5.2 契約書の必須記載事項（第19条第1項）

| # | 記載事項 | システムでの管理方法 |
|---|---------|-------------------|
| 1 | 工事内容 | 工事概要テキスト、工事種別選択 |
| 2 | 請負代金の額 | 金額フィールド、消費税管理 |
| 3 | 工事着手の時期及び工事完成の時期 | 日付フィールド、工期自動計算 |
| 4 | 工事を施工しない日又は時間帯の定めをするときは、その内容 | 非施工日時設定 |
| 5 | 請負代金の全部又は一部の前金払又は出来形部分に対する支払の定めをするときは、その支払の時期及び方法 | 支払条件管理 |
| 6 | 当事者の一方から設計変更又は工事着手の延期若しくは工事の全部若しくは一部の中止の申出があった場合における工期の変更、請負代金の額の変更又は損害の負担及びそれらの額の算定方法に関する定め | 変更条項テンプレート |
| 7 | 天災その他不可抗力による工期の変更又は損害の負担及びその額の算定方法に関する定め | 不可抗力条項管理 |
| 8 | 価格等の変動若しくは変更に基づく請負代金の額又は工事内容の変更 | スライド条項管理 |
| 9 | 工事の施工により第三者が損害を受けた場合における賠償金の負担に関する定め | 賠償条項管理 |
| 10 | 注文者が工事に使用する資材を提供し、又は建設機械その他の機械を貸与するときは、その内容及び方法に関する定め | 支給材・貸与機管理 |
| 11 | 注文者が工事の全部又は一部の完成を確認するための検査の時期及び方法並びに引渡しの時期 | 検査・引渡し管理 |
| 12 | 工事完成後における請負代金の支払の時期及び方法 | 支払条件管理 |
| 13 | 工事の目的物の瑕疵を担保すべき責任又は当該責任の履行に関して講ずべき保証保険契約の締結その他の措置に関する定めをするときは、その内容 | 瑕疵担保条項管理 |
| 14 | 各当事者の履行の遅滞その他債務の不履行の場合における遅延利息、違約金その他の損害金 | 遅延損害金条項管理 |
| 15 | 契約に関する紛争の解決方法 | 紛争解決条項管理 |

### 5.3 下請契約の規制

| 規制事項 | 条文 | システム対応要件 |
|---------|------|---------------|
| 一括下請負の禁止 | 第22条 | 下請比率チェック、実質的関与確認チェックリスト |
| 下請代金の支払 | 第24条の3 | 支払期日管理（引渡し後50日以内）、遅延アラート |
| 特定建設業者の下請代金支払義務 | 第24条の6 | 下請代金支払期限の自動計算、監視 |
| 施工体制台帳の作成 | 第24条の8 | 施工体制台帳の自動生成、更新管理 |

## 6. 技術者制度（第25条の27〜第27条の18）

### 6.1 技術者の配置義務

| 技術者種別 | 配置要件 | システム管理要件 |
|-----------|---------|---------------|
| 主任技術者 | 全ての建設工事に配置（第26条第1項） | 工事別技術者配置管理、資格確認 |
| 監理技術者 | 特定建設業者が元請で4,500万円以上の下請契約を締結する場合（第26条第2項） | 配置要否の自動判定、監理技術者資格者証管理 |
| 監理技術者補佐 | 監理技術者の補佐（第26条の3） | 補佐技術者の管理、特例監理技術者の二現場兼務管理 |
| 専任の技術者 | 公共工事または民間工事で請負代金4,000万円以上（建築は8,000万円以上） | 専任性チェック、兼務状況管理 |

### 6.2 技術者管理機能

| 管理機能 | 詳細 |
|---------|------|
| 資格情報管理 | 保有資格（1級/2級施工管理技士、技術士等）の登録・有効期限管理 |
| 配置管理 | 工事ごとの技術者配置状況管理、配置可能技術者の検索 |
| 専任性チェック | 複数工事への重複配置検出、専任要件の自動判定 |
| 資格者証管理 | 監理技術者資格者証の登録・有効期限管理（5年更新） |
| CPD管理 | 継続教育単位の記録・集計 |
| CCUS連携 | 建設キャリアアップシステムとのデータ連携 |
