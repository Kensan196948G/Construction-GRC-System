## 1. 目的

本文書は、Construction-GRC-Systemが準拠する全法令・規格間のクロスリファレンス（相互参照）テーブルを定義する。重複要件の統合管理方法を明確化し、効率的なコンプライアンス管理を実現することを目的とする。

## 4. 重複要件の統合管理

### 4.1 重複要件の特定

以下の管理領域で、複数の法令・規格間に重複する要件が存在する。

| 重複領域 | 重複する規格/法令 | 統合管理方法 |
|---------|----------------|------------|
| **記録管理** | ISO27001(A.5.33) + 建設業法(第19条) + 労安法(第103条) | 統合記録管理モジュールで一元管理、法定保存期間の最長値を適用 |
| **教育・訓練** | ISO27001(A.6.3) + 品確法(第8条の3) + 労安法(第59条) | 統合教育管理モジュールで一元管理、教育種別を横断的に追跡 |
| **リスクアセスメント** | ISO27001(A.5.1) + NIST CSF(ID.RA) + 労安法(第28条の2) | 統合リスク管理モジュール、情報セキュリティリスクと労働安全リスクを統合管理 |
| **サプライチェーン管理** | ISO27001(A.5.19-22) + NIST CSF(GV.SC) + 建設業法(第22条,24条の8) | 統合サプライヤー管理、セキュリティ評価と法的要件を一体管理 |
| **インシデント管理** | ISO27001(A.5.24-28) + NIST CSF(RS) + 労安法(第100条) | 統合インシデント管理、セキュリティと安全のインシデントを一元管理 |
| **監査・レビュー** | ISO27001(A.5.35) + NIST CSF(GV.OV) + 建設業法(第28条) | 統合監査管理、コンプライアンス監査を一元管理 |
| **個人情報保護** | ISO27001(A.5.34) + 個人情報保護法 + 労安法(第104条) | 統合PII管理、個人情報・健康情報を一元管理 |

### 4.3 法定保存期間の統合管理

記録の保存期間は、該当する全法令の要件のうち最も長い期間を適用する。

| 記録種別 | ISO27001 | 建設業法 | 品確法 | 労安法 | 個人情報保護法 | 適用保存期間 |
|---------|----------|---------|--------|--------|-------------|------------|
| 契約書類 | 規定なし（ISMSの範囲で定める） | 5年 | - | - | - | **5年以上** |
| 施工記録 | 規定なし | 5年（帳簿） | 10年（検査記録） | - | - | **10年以上** |
| 安全衛生教育記録 | 規定なし | - | - | 3年 | - | **3年以上** |
| 健康診断結果 | 規定なし | - | - | 5年（一般）/ 30年（特殊）/ 40年（石綿） | - | **最大40年** |
| 作業環境測定記録 | 規定なし | - | - | 3年（一般）/ 30年（特定化学物質） | - | **最大30年** |
| 労災報告書 | 規定なし | - | - | 5年 | - | **5年以上** |
| 個人情報取扱記録 | 規定なし | - | - | - | 3年 | **3年以上** |
| 監査ログ | ISMSで定める | - | - | - | - | **5年以上（推奨）** |
| セキュリティインシデント記録 | ISMSで定める | - | - | - | - | **5年以上（推奨）** |
