## 1. 概要

### 1.1 目的

本文書は、Construction-GRC-Systemが準拠するISO27001:2022、NIST CSF 2.0、建設業法、品確法（公共工事品質確保法）、労安法（労働安全衛生法）の全クロスリファレンステーブルを提供する。各規格・法令間の対応関係を明確化し、重複要件の統合管理方針を定める。

## 4. 重複要件の統合管理方針

### 4.1 統合管理の基本原則

| 原則 | 説明 |
|------|------|
| 最も厳格な基準の採用 | 複数規格・法令で異なる要件がある場合、最も厳格な基準を採用する |
| 一元管理 | 同一テーマの管理活動は1つのプロセスで統合管理する |
| 記録の共有 | 複数の要件を満たす記録は一度の記録作成で共有する |
| エビデンスの再利用 | 1つの管理活動のエビデンスを複数の監査・審査で活用する |

### 4.2 重複要件の統合マッピング
