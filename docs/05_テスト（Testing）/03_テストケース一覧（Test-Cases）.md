## 1. 概要

本文書は、Construction-GRC-Systemの全テストケースをID付きで一覧化したものである。各テストケースにはテスト条件、入力データ、期待結果、優先度を記載する。

### テストケースID体系

```
TC-<モジュール>-<機能番号>-<連番>

例: TC-RSK-001-001
```

| モジュールコード | モジュール名 |
|----------------|------------|
| AUTH | 認証・認可 |
| RSK | リスク管理 |
| CMP | コンプライアンス管理 |
| CTL | 統制管理 |
| AUD | 監査管理 |
| RPT | レポート管理 |
| NTF | 通知機能 |
| CMN | 共通機能 |

### 優先度

| 優先度 | 説明 |
|--------|------|
| P1 | 必須（リリースブロッカー） |
| P2 | 重要（リリース前に実施） |
| P3 | 通常（可能な限り実施） |
| P4 | 低（時間があれば実施） |
